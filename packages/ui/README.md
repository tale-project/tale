# @tale/ui

Shared Tale React component library

```bash
bun run --filter @tale/ui typecheck
bun run --filter @tale/ui test
bun run --filter @tale/ui storybook
```

Source-only React component package — consumers import directly from `@tale/ui/<subpath>`. Ships a Tailwind preset (`@tale/ui/tailwind-preset`) and a base stylesheet (`@tale/ui/globals.css`).
