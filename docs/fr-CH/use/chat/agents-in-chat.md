---
title: Utiliser des agents dans le chat
description: Choisis des agents spécialisés depuis la zone de saisie pour changer la façon dont Tale répond.
---

Un agent est une version de l'IA adaptée à un usage précis — agent support, agent sales, agent recherche, etc. Chaque agent a ses propres instructions, son accès aux connaissances et ses permissions d'outils. Tu choisis quel agent gère une conversation depuis le **sélecteur d'agent** de la zone de saisie.

## Changer d'agent

1. Clique l'icône **bot** en bas à gauche de la zone de saisie.
2. Une liste s'ouvre avec l'agent par défaut en tête, suivi des agents personnalisés construits par ton équipe.
3. Clique l'agent voulu. La barre de titre de la conversation montre l'agent actif.

Chaque conversation garde son agent sélectionné. Un nouveau chat repart sur l'agent par défaut.

## Démarreurs de conversation

Quand tu commences une nouvelle conversation avec un agent qui a des **démarreurs** configurés, une rangée de suggestions cliquables apparaît. Clique-en une pour l'envoyer comme premier message — c'est plus rapide que de taper, et c'est un bon moyen de découvrir à quoi sert l'agent.

## Pourquoi changer d'agent ?

- **Meilleures réponses sur des sujets précis** — un agent support pointé uniquement sur les documents support ne se laisse pas distraire par des connaissances hors sujet.
- **Outils restreints** — un agent de recherche en lecture seule ne peut rien modifier, tu peux poser des questions exploratoires sans risquer de changements accidentels.
- **Ton différent** — les agents peuvent être configurés avec des voix, des formats de sortie et des niveaux de rigueur différents.

## Transferts (délégation)

Certains agents sont configurés pour **déléguer** à un autre agent quand le sujet dérive. Si tu poses une question de facturation à un agent support général, il peut transférer automatiquement à un agent billing spécialisé. Dans ce cas, la conversation affiche une courte note expliquant le basculement.

## Créer ton propre agent

Avec les permissions Editor ou plus, tu peux construire des agents pour ton équipe. Voir [Créer un agent](/fr-CH/build/agents/create).
