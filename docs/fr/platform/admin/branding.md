---
title: Branding
description: Personnalise le nom de l'application, le logo, la favicon et les couleurs de marque pour ton organisation.
---

Branding permet aux admins de personnaliser l'apparence de Tale pour tous dans l'organisation. Les changements s'appliquent à l'onglet du navigateur, la barre de navigation, les e-mails et les en-têtes de notification.

Options disponibles dans **Paramètres > Branding** :

| Option                   | Description                                                                    |
| ------------------------ | ------------------------------------------------------------------------------ |
| **Nom de l'application** | remplace "Tale" dans l'onglet du navigateur et l'en-tête.                      |
| **Logo**                 | image affichée dans la barre de navigation. Remplace le logo Tale par défaut.  |
| **Favicon**              | icône dans l'onglet du navigateur. Images séparées pour modes clair et sombre. |
| **Couleur de marque**    | couleur principale pour les boutons et états actifs.                           |
| **Couleur d'accent**     | couleur secondaire pour les highlights et badges.                              |

Les changements s'appliquent immédiatement pour tous les utilisateurs — pas besoin de recharger sur les autres clients.

## Logo et favicon

- Pour le logo, téléverse PNG, JPEG ou SVG. SVG est recommandé pour un rendu net à toutes les tailles.
- Les favicons doivent être des PNG carrés d'au moins 512 × 512 px. Tale les redimensionne automatiquement pour les différentes tailles de navigateur.
- Les variantes mode clair/mode sombre permettent d'utiliser un logo foncé sur fond clair pour le clair et un logo clair sur fond foncé pour le sombre.

## Couleurs

Les couleurs sont entrées en codes hex. La plateforme impose un ratio de contraste 4.5:1 par rapport au fond — si une couleur choisie passe sous, un avertissement d'accessibilité apparaît et le sélecteur propose une alternative proche qui respecte le contraste.

## Réinitialiser

Clique l'icône reset à côté d'un champ pour revenir à la valeur Tale par défaut. Réinitialiser nom et logo ensemble restaure le branding Tale par défaut.
